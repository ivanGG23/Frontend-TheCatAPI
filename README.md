# frontend-TheCatAPI

Frontend en HTML, CSS y JavaScript puro con Tailwind CSS.

## Requisitos
- El backend debe estar corriendo en http://localhost:4000
- Live Server (VS Code) o Docker

## Cómo correr

### Con Live Server
1. Abre la carpeta en VS Code
2. Click derecho en index.html → Open with Live Server

### Con Docker
Usa el repo de infraestructura.

## Páginas

| Página | Descripción |
|--------|-------------|
